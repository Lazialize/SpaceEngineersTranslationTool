【はじめに】
Space Engineers翻訳ツールをダウンロードしていただきありがとうございます。
当ツールはSpace Engineersの日本語化を自動化すべく開発されました。

【日本語フォントについて】
日本語フォントはSteamワークショップの"Japanese Font Pack / 日本語フォント追加MOD"より拝借しております。
こちらのフォントが最新のバージョンに対応しておりませんでしたので、こちらで修正および適用させていただきました。
公開の停止要請、あるいはModの更新によって対応し次第、こちらのファイルからフォントデータを削除させて頂きます。

Japanese Font Pack / 日本語フォント追加MOD
http://steamcommunity.com/sharedfiles/filedetails/?id=442628194


【使い方】

1. 「Space Engineers Translation Tool(.exe)」をダブルクリックして当ツールを起動してください。

2. 起動したら「翻訳ファイルダウンロード」をクリックして翻訳ファイルのダウンロードを行ってください。

  2-1. 最終更新日が更新されればダウンロードは成功しています。

3. ダウンロードが終わったら「翻訳を適用」ボタンをクリックして導入が終わるまで待機してください。

  3-1. 導入が正常に終了した場合、ウィンドウ左下に適用の結果の文字が表示されます。

  3-2. 設定タブからアイテム名・ブロック名を翻訳するかどうかが選択できます。

4. 3の手順が終了しても、文字化けを起こしてしまうので次に「フォントの適用」をクリックしてください。

Option. 英語に戻したい場合は、「英語に戻す」をクリックしてください。
※「英語に戻す」は正確には、翻訳サイトに登録してある英語原文が適用されます。
